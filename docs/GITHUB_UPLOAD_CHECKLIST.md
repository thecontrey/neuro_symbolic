# GitHub Upload Checklist

1. Create a public GitHub repository.
2. Upload all files from this package preserving folder structure.
3. Add repository URL to the paper:
   - Supplementary Materials;
   - Data Availability Statement;
   - Appendix C.
4. Check that no raw contracts, names, signatures, addresses, commercial party identifiers, or confidential clauses are included.
5. Add a release tag, for example `v1.0-paper-submission`.
6. Optionally archive the release in Zenodo and insert DOI.
