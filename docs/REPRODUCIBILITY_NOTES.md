# Reproducibility Notes

The comparison against end-to-end LLM+RAG, PDL, simpler symbolic baselines, and the proposed
LLM-to-NMR pipeline is reported descriptively in the paper. Formal McNemar/Bonferroni paired
significance testing is not claimed in the current manuscript version.

The package includes scenario-level predictions and aggregate tables so that descriptive metrics,
bootstrap intervals, and proof-graph checks can be inspected and rerun.
