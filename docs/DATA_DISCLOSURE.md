# Data Disclosure and Confidentiality

This package excludes raw contractual wording, original party names, commercial terms, and any
information that could identify contracting parties.

The released material is an anonymized scenario-level representation. It is designed to preserve
the legal-logical structure required for method verification:
- source-unit identifiers;
- facts;
- strict and defeasible rules;
- defeaters;
- priorities;
- attack graphs;
- proof objects;
- gold labels;
- predictions;
- validation logs;
- aggregate metrics.

The package supports independent inspection of the symbolic reasoning layer, proof-checking
protocol, baseline comparisons, and aggregate evaluation without disclosing confidential source contracts.
